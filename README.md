# 🏏 Cricket Analytics Dashboard
A machine learning-powered cricket analytics dashboard that predicts IPL match scores in real-time and provides interactive visual insights such as run rate, win probability, and venue-based performance using Dash and Plotly.

## 🚀 Features
- Live match simulation
- Score prediction using ML
- Run rate analysis
- Venue comparison
- Interactive dashboard UI

## 🛠 Tech Stack
- Python
- Dash
- Plotly
- Scikit-learn
- Pandas

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
